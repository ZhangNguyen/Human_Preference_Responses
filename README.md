# Single Model Backend

Space này chạy 1 model duy nhất.

## Variables

- `MODEL_ROLE=base` cho model gốc
- `MODEL_ROLE=finetuned` cho model finetune
- `MODEL_ID=your-username/your-model` nếu muốn load model trực tiếp từ Hugging Face Hub
